## Extra practice



1. Write a function to interlace 2 strings:
```java
interlace("abc", "123") -> "a1b2c3"
interlace("bed", "ras") -> "breads"
```

2. Write a function to find the position of the 2nd "a" in a string: 
```java
find2nd("banana") -> 3
find2nd("happy birthday") -> 12
```

3. Write a function to add a specified number of "na" strings to "ba":
```java
addNa(0) -> "ba"
addNa(1) -> "bana"
addNa(2) -> "banana"
addNa(3) -> "bananana"
```

4. Write a function to calculate the sum of powers of 2:
```java
sumPowers(0) -> 0
sumPowers(1) -> 0+1 = 1
sumPowers(2) -> 0+1+2 = 3
sumPowers(3) -> 0+1+2+4 = 7
sumPowers(4) -> 0+1+2+4+8 = 15 
```
